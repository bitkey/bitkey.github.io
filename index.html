The secure open source live USB key solution specifically designed for bitcoin

Overview

BitKey is a portable device that a computer can boot from, containing a prefabricated independent operating system environment which is engineered from the ground up to prioritize security while maximizing usability, in order to provide a safe, reliable and easy to use practical platform for high risk applications such as bitcoin transactions.

Booting the Live OS (red)

You boot the OS into RAM from CD (preferably) or from a red USB which you aren't supposed to use for anything else to prevent malicious modification.

Booting from CD is more secure because it's physically read-only and you don't have to worry about BadBIOS style attacks against the USB stack.

In the future we'd like to explore making the red USB physically read-only (ROM chip). Currently there's no secure way to do that with commodity USB hardware so we'll be working around that at a software level.

Boot modes

BitKey is a single image supporting two use cases via three modes of operation. At boot time, you select the desired mode from the bootloader menu:

Mode Wallet Transactions Security Convenience
cold-offline create sign high less
cold-online watch prepare high less
hot-online create & watch prepare & sign medium more

Data storage (black)

In addition to the BitKey device, you'll need a black USB key which will serve as storage for encrypted wallet files, unencrypted master public key for watch-only mode, and optionally as a data shuttle for unsigned and signed transactions between cold online and offline modes.

Usage: Hot (online)

Fully functional bitcoin client. This mode is appropriate for relatively small-medium bitcoin transactions.

  • Insert BitKey device and power on PC
  • Select hot-online from the bootmenu
  • Once the OS has loaded, remove BitKey device
  • Insert black data storage USB device
  • Open Electrum
    • Enter passhprase for secure encrypted storage
    • Select create wallet
    • Write down mnemonic seed on paper (store in lockbox)
    • Prepare and sign bitcoin transactions...
  • Close Electrum
  • Remove black data storage USB device
  • Shutdown BitKey

    • Features

    Open Source

    BitKey is open source and hosted on Github. Available for peer-review as well as improvement by the bitcoin community.

    Encrypted Wallet Storage

    Wallet files are stored in a LUKS encrypted loopback filesystem. During creation, estimated passphrase strength such as entropy and crack time is displayed.

    Deterministic builds (planned)

    We will be improving our already open source build chain to support deterministic bit-for-bit builds, to ensure the build machines have not been compromised, as well as enabling the community to verify integrity, and create a community based signed chain of trust for each released image build.

    Lightweight Debian Live OS

    BitKey boots live (no installation required), and copies itself into RAM for both speed as well as security of the boot media.

    Included software

    Electrum bitcoin client, incognito chromium, bitaddress, zxcvbn. File manager, network and wireless manager. Auto-mounting USB storage devices support.

    Multi layered security (planned)

    We will be implementing exotic security mechansims such as mandatory access control (MAC), PAX, trusted path execution (TPE), position independent code address space layout randomization (PIE-ASLR), discretionary access control (DAC), jails, exploit countermeasures (ECM) and raw IO/Memory protections.

    About BitKey

    BitKey is a project by the core developers of TurnKey Linux. They are bitcoin enthusiasts with security backgrounds who needed a secure and easy way to participate in the bitcoin world.

    Using their expertise from TurnKey Linux, they created a custom solution for themselves, which evolved into the open source BitKey project.

    Other bitcoin related projects by the TKL core devs include Block Explorer - the open source web tool for exploring the bitcoin blockchain.
